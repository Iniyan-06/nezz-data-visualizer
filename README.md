# Nezz Data Visualizer

## 1. What it does
Nezz is a high-performance, client-side data exploration tool designed for privacy-first analysis. It uses a 3-step pipeline to clean CSV/JSON data, analyze trends via a local AI model (**Qwen 3 0.6B**), and generate dynamic Chart.js visualizations. It features automated anomaly detection, smart chart selection, and a proprietary "Wow Gap" efficiency score.

## 2. How to run it
Run the following command in your terminal to initialize the environment and start the dashboard:

```bash
chmod +x setup.sh && ./setup.sh
```
*Note: Ensure [Ollama](https://ollama.com) is installed and running on your system.*
